2.3 ¿Qué significa que un puerto esté abierto?

Cuando un puerto está abierto, significa que hay un servicio o programa en tu computadora que está escuchando conexiones externas a través de ese puerto.
Peligros: Si se encuentran puertos abiertos sin saberlo, puede ser peligroso ya que los atacantes pueden intentar entrar por ellos a través de vulnerabilidades conocidas.
Ejemplos de ataques a través de puertos abiertos:

Ransomware

Es un tipo de malware que cifra los datos de una víctima, impidiéndole acceder a sus archivos o sistemas hasta que se pague un rescate.

Una de las variantes más significativas de ransomware fue "WannaCry", el cual se propagó rápidamente en mayo de 2017 aprovechando una vulnerabilidad en Windows. Se difundió a través de redes y causó interrupciones generalizadas, afectando a muchas empresas y a usuarios individuales.





Blaster Worm


El gusano se replica a sí mismo y se propaga a otras computadoras conectadas a la misma red, aprovechando vulnerabilidades en el servicio DCOM (Modelo de Objetos Componentes Distribuidos) de Microsoft para infectar computadoras que ejecutan sistemas operativos Windows XP y Windows 2000.

3.1 ¿Por qué es importante que un puerto esté abierto para que un servicio funcione?

Cuando una computadora o servidor ofrece un servicio (como una página web, un servidor de archivos), necesita una forma de comunicarse con otros dispositivos que quieren usar ese servicio. Si el puerto está cerrado, los datos que llegan desde Internet o desde otra computadora no pueden entrar. Es como si hubiera una pared bloqueando esa puerta.

3.2 ¿Por qué no todos los puertos están abiertos?

Si todos los puertos estuvieran abiertos, sería muy fácil para atacantes entrar a la computadora o red y causar problemas, como robar información o controlar el sistema. Por eso, los sistemas operativos mantienen la mayoría de los puertos cerrados o filtrados para proteger la seguridad. Solo abren los puertos necesarios para los servicios que realmente se usan, como un servidor web (puerto 80) o un servidor de correo (puerto 25).

3.3 Si un firewall está activo, ¿qué efecto tiene en el escaneo?

Un firewall actúa como una barrera de seguridad entre tu computadora y el exterior. Cuando está activo, controla qué conexiones entrantes o salientes se permiten o bloquean. Por eso, si hacés un escaneo de puertos con un firewall activo, muchos puertos pueden aparecer como cerrados o filtrados, aunque realmente haya servicios escuchando en ellos. Esto sucede porque el firewall bloquea o no responde a las solicitudes del escáner para proteger el sistema. En resumen, un firewall puede ocultar puertos abiertos, dificultando que un atacante vea qué servicios están disponibles en tu equipo.

4.1 ¿Por qué un atacante querría escanear puertos?

Un atacante escanea puertos para descubrir qué servicios o programas están activos en una computadora o red. Esto es importante porque cada servicio abierto puede tener vulnerabilidades o fallas que el atacante puede aprovechar para entrar al sistema. Al saber qué puertos están abiertos, el atacante puede planear ataques específicos, como intentar entrar por un puerto que tiene una versión antigua o mal protegida de un programa.

4.2 Si tienes un servidor web en tu máquina (puerto 80) y está abierto, ¿qué implicaciones de seguridad tiene?

Tener el puerto 80 abierto significa que estás ejecutando un servidor web en tu computadora, accesible para cualquiera desde internet o desde la red local. Esto puede ser un riesgo si el servidor web tiene vulnerabilidades o no está bien configurado. Los atacantes pueden intentar explotar errores del servidor y acceder a datos privados. Por eso, si el puerto 80 está abierto, es muy importante:
◉ Mantener el servidor web actualizado.
◉ Usar contraseñas seguras.
◉ Configurar correctamente los permisos.
◉ Implementar medidas de seguridad como firewalls o sistemas de detección de intrusos.

4.3 ¿Qué hardware o software ayuda a 'cerrar' o 'filtrar' puertos no deseados?

Los firewalls son las principales herramientas que permiten cerrar o filtrar puertos no deseados y así proteger una computadora o red de posibles ataques.

Existen dos tipos principales de firewalls:

◉ Firewall de Software: Son programas instalados en la computadora o servidor. Permiten controlar qué conexiones se permiten y cuáles se bloquean. Ejemplos conocidos son: Windows Defender Firewall, ZoneAlarm o Comodo Firewall.

◉ Firewall de Hardware: Son dispositivos físicos que se colocan entre la red interna y el internet. Protegen toda una red de computadoras filtrando el tráfico antes de que llegue a los dispositivos. Algunos ejemplos son: routers con firewall incorporado, Cisco ASA o equipos Fortinet.
